# FastAPI-MongoDB-NotesApp
A basic web application for saving and displaying notes/TODOs using FastAPI and MongoDB in python.

Tech-Stack involved - 
1. FastAPI (GET and POST requests)
2. Python
3. Pydantic
4. PyMongo
5. MongoDB
6. Bootstrap
7. HTML
